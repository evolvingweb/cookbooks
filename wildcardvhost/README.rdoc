= DESCRIPTION:
This will configure apache to listen on http://*.fqdn and http://*.nodename and serve requests
using /var/shared/sites/*/site as the root.

For example,

http://coolproject.coolhost -> /var/shared/sites/coolproject/site
http://coolproject.coolhost.example.com -> /var/shared/sites/coolproject/site

= REQUIREMENTS:
Requires apache2 recipe.
